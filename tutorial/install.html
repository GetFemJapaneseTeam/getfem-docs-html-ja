インストール方法¶

GetFEM++ は linux (ubuntu) 上で開発しされているため linux 、特に Debian ベースのディストリビューション (Debian/Ubuntu/Mint) では、インストールは簡単です。しかし、Mac os X および Windows 上の他の linux ディストリビューションにも GetFEM++ をインストールすることができます。GetFEM++ をコンパイルするためにソースから、最近の C++ コンパイラ (C++ 11 標準をサポート) と最新バージョンの python 2 が必要です。

GetFEM++ の他のライブラリの主な依存は

  • git クライアント、automake、autoconf、および libtool は、git バージョンからバイナリをビルドして最新の変更を取得する場合に使用します。

  • Python インターフェイスを構築する場合は、Python 開発ファイル (Python.h など) と numpy と scipy パッケージが必要です。

  • GetFEM++ と一緒に配布されされている SuperLU の代わりに使用する場合、直列 MUMPS パッケージ (疎行列の直接ソルバー) が必要です。

  • MPI で並列化されたバージョンの GetFEM++ を使用したい場合は、 METIS と MPI4PY の並列パッケージが必要です。

  • メッシュ生成と架空領域アプリケーションのための qhull パッケージ

  • BLAS と LAPACK パッケージ

GetFEM++ C++ ライブラリは、独自に構築することも、Python、Scilab、および/または Matlab インターフェイスと一緒にビルドするこもできます。

また、対応するパッケージ管理システムを使用して linux ディストリビューションに Getfem の安定版バージョンをインストールすることもできます。

Getfem C++ ライブラリをビルドする方法の詳細については、 ダウンロードしてインストールする を参照してください。
