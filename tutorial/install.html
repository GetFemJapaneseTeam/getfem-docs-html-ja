インストール方法¶

以降 | gf |linux 上で開発し (ubuntu の)、インストールは linux 上で、特に debian ベースのディストリビューション (debian/ubuntu の/ミント) で簡単です。しかし、| gf |Mac os X および Windows 上の他の linux ディストリビューションにもインストールできます。ためにコンパイルする | gf |ソースからは、最近の c++ コンパイラ (c++ 11 標準をサポート) と python 2 の最新バージョンが必要です。

の主な依存性 |Gf |他のライブラリでは

  • svn クライアント、automake、autoconf、libtool は、最新の変更を取得するために svn のバージョンからバイナリを構築したい場合。

  • python 開発ファイル (python. h など) と、| np |と | sp |python インターフェイスを構築する場合は、パッケージを選択します。

  • シーケンシャルおたふく風邪のパッケージ (疎行列の直接ソルバー) あなたは SuperLU のバージョンの代わりにそれを使用する場合は | gf | と一緒に配布されます。

  • パラレルおたふく風邪、MPI4PY パッケージ | gf の MPI の並列化されたバージョンを ue したい場合は |

  • メッシュ生成および架空ドメインアプリケーション用の qhull パッケージ

  • ブラスと LAPACK パッケージ

GetFEM++ C++ library can be build on its own or together with the Python, Scilab and/or Matlab interface.

また、対応するパッケージ管理システムを使用して、Getfem の安定版を linux ディストリビューションにインストールすることもできます。

Getfem C++ ライブラリをビルドする方法に関するより具体的な情報については、「ダウンロードしてインストールする <.../download.html > ‘ _ ページを表示します。
