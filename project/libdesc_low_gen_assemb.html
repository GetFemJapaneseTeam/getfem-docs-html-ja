The low-level generic assembly module in GetFEM++¶

Description¶

First version of the generic assembly. Base on tensor reduction. Not very convenient for nonlinear terms. The high-level generic assembly have to be prefered now.

Files¶

File(s) Description
getfem_mat_elem_type.h and:file:` getfem_mat_elem_type.cc Defines bes type for components of an elementary matrix.
getfem_mat_elem.h and:file:` getfem_mat_elem.cc Describes an compute elementary matrices.
getfem_assembling_tensors.h and:file:getfem_assembling_tensors.cc Performs the assembly.
getfem_assembling.h Various assembly terms (linear elasticity, generix elliptic term, Dirichlet condition ...

State¶

Stable.

Perspectives¶

Will not evolve since the efforts are now focused on the high-level generic assembly.
