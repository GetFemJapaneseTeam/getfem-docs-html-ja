GetFEM++ のFEM記述¶

このセクションの目的は、 GetFEM++ でFEM記述を簡単に紹介することです。主に残りの文書で使用される表記法（要素の定義、参照要素、幾何学的変換、幾何学的変換の勾配...）を修正するためです。

凸包構造¶

有限要素法は、要素と呼ばれる小さな凸包領域に定義されています。有限要素法を定義できる最も単純な要素は、セグメント（次元1のシンプレックス）、他には三角形、四面体（次元2と3のシンプル）、プリズム、平行六面体などです。 GetFEM++ では、オブジェクトの bgeot::convex_structure は、 bgeot_convex_structure.h ファイルで定義されています。

それは頂点の座標ではなく凸面の構造だけを記述します。このタイプの構造は同じタイプの凸包部を記述する必要はないので、この構造はそれ自身で操作することはできません。このような記述子上のポインタは、 bgeot::pconvex_structure 型で宣言しなければなりません。

次の関数は、通常の型の要素の記述子にポインタを与えます。

bgeot::simplex_structure(dim_type d)¶

次元 d のシンプレックスの記述

bgeot::parallelepiped_structure(dim_type d)¶

次元 d の平行六面体の記述

bgeot::convex_product_structure(bgeot::pconvex_structure p1, bgeot::pconv$

p1 と p2 の直積の記述

bgeot::prism_P1_structure(dim_type d)¶

次元 d のプリズムの記述

たとえば、正方形の記述が必要な場合、以下のように等価的に表現できます。

p = bgeot::parallelepiped_structure(2);

または

p = bgeot::convex_product_structure(bgeot::simplex_structure(1),
                                    bgeot::simplex_structure(1));

記述子には特に頂点の数( nb_points() ）のための面の数（ p->nb_faces() ）と凸部の次元（ p->dim() ）が含まれます。他の情報は、各面の頂点の数、面の記述、およびより基本的な（幾何学的変換の記述に使用される）記述への最終的な参照です。

通常の要素

参照の凸包¶

参照の凸包は特定の実要素、すなわち頂点のリストを伴う凸の構造です。これは有限要素法が定義されている特定の要素を記述します。ファイル bgeot_convex_ref.h 内のオブジェクト bgeot::convex_of_reference がそれを記述します。ライブラリは、凸型の各タイプについて1つの記述しか保持しません。したがって、操作されるのは、ディスクリプタ上のタイプ bgeot::pconvex_ref のポインタです。

次の関数は記述を構築します。

bgeot::simplex_of_reference(dim_type d)¶

次元 d の参照のシンプレックスの記述。

bgeot::simplex_of_reference(dim_type d, short_type k)

次数 k の Lagrange グリッドの次元 d を参照するシンプレックスの記述

bgeot::convex_ref_product(pconvex_ref a, pconvex_ref b)¶

2つの参照凸包の直積の記述。

bgeot::parallelepiped_of_reference(dim_type d)¶

次元 d を参照する平行六面体の記述。

形状関数型¶

ほとんどの場合、有限要素法の形状関数は、少なくとも参照の凸包の多項式です。しかし、要素の他のタイプを持って与えられる可能性もあります。他の種類の基底関数 (区分多項式、補間曲線ウェーブレットなど) を定義することができます。

To be used by the finite element description, a shape function type must be able to be evaluated on a point (a = F.eval(pt), where pt is a base_node) and must have a method to compute the derivative with respect to the ith variable (F.derivative(i)).

現時点では、ファイル bgeot_poly.h および bgeot_poly_composite.h では、多項式と区分多項式のみが定義されています。

幾何学的変換の記述子に対するポインタは、ファイル bgeot_geometric_trans.h で定義されている次の関数によって取得できます。

bgeot::pgeometric_trans pgt = bgeot::geometric_trans_descriptor("name of trans");

ここで、 "name of trans" を次のリストの中から選択することができます。

  • "GT_PK(n,k)"

    次元 n および次数 k のシンプレックス変換の記述 (ほとんどの場合、次数1が使用されます)。

  • "GT_QK(n,k)"

    次元 n および次数 k の直方体変換の記述。

  • "GT_PRISM(n,k)"

    次元 n および次数 k のプリズム変換の記述。

  • "GT_PRODUCT(a,b)"

    2つの変換 a と b の直積の記述。

  • "GT_LINEAR_PRODUCT(a,b)"

    Description of the direct product of the two transformations a and b keeping a linear transformation (this is a restriction of the previous function). This allows, for instance, to use exact integrations on regular meshes with parallelograms.
