モデルモジュール¶

説明¶

モデル (変数をリンクする変数、データ、および数式の項) について説明します。

ファイル¶

ファイル

説明

getfem_models.h と getfem_models.cc

オブジェクトモデル、内部および標準ブリック (線形弾力性、汎用楕円ブリック、ディリクレ境界条件) を定義します。

getfem_model_solvers.h と getfem_model_solvers.cc

モデルオブジェクトの標準ソルバーを定義します。

getfem_contact_and_friction_common.h と getfem_contact_and_friction_common.cc

変形体の接触/摩擦条件のための共通アルゴリズム

getfem_contact_and_friction_integral.h と getfem_contact_and_friction_integral.cc

小型摺動接点/一体型の摩擦ブリック。

getfem_contact_and_friction_large_sliding.h と getfem_contact_and_friction_large_sliding.cc

大型摺動接点/摩擦ブリック。

getfem_contact_and_friction_nodal.h と getfem_contact_and_friction_nodal.cc

小摺動節接触/摩擦ブリック。
getfem_Navier_Stokes.h

Navier-Stokesブリックの試み。改善される。

getfem_fourth_order.h と getfem_fourth_order.cc

Bilaplacian と Kirchhoff-Love プレートレンガ

getfem_linearized_plates.h と getfem_linearized_plates.cc

Mindlin-Reissner プレートブリック

getfem_nonlinear_elasticity.h と getfem_nonlinear_elasticity.cc

大変形弾性ブリック。

getfem_plasticity.h と getfem_plasticity.cc

可塑性ブリック。

状態¶

一定の進化は、nex のモデルが含まれます。

展望¶

より多くの板、荷重およびシェルブリック、大変形の可塑性、...
