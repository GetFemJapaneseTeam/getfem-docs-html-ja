Integ モジュール¶

説明¶

立体求積法モジュールは、参照要素の数値積分法へのアクセスを提供します。それはまた、いくつかの正確な積分方法へのアクセスを与えるため、実際にはそれだけでいくつかの立体求積法式が含まれていません。ただし、厳密な積分法は多項式要素とアフィン幾何学的変換に対してのみ使用できます。これは厳密な積分方法が広く使用されない理由の説明となります。立体求積法式の説明は、 getfem_integration.h または GetFEM++ のディレクトリ cubature の説明ファイルを介してファイル内で直接実行されます。新しい立体求積法式の追加は非常に単純であり、それはそれが定義されている要素を参照し、ファイル内のガウス点のリストと、このディレクトリに追加するだけで十分です。さらに、ドメインの境界に定義された用語を統合するために、その説明には、要素の各面で同じ順序のメソッドへの参照も含まれている必要があります。

ファイル¶

ファイル

説明

: ファイル: ‘ getfem_integration ‘ と: ファイル: ‘ getfem_integration ‘ と: ファイル: ‘ getfem_integration_composite ‘

積分方法の構造、基本的な積分方法、積分方法および複合方法のプロダクト。

: ファイル: ‘ getfem_im_list ‘

 file generated by cubature/make_getfem_list with the integration methods defined in cubature directory. This gives the possibility to define a new integration method just listing the Gauss points and weigth in a text file.

状態¶

このモジュールは、プロジェクトの現在のニーズを満たしており、安定していると考えられています。使用可能な立体求積法式のリストは、ref: ‘ ud-付録 b ‘ で指定されています。

視点¶

今のところ変更は必要ありません。新しい立体求積法式 (descritption ファイルの形式) を追加する方法を完全に説明するために、ドキュメントで努力を行うことができます。
