レベルセットモジュール¶

説明¶

1つまたは複数のレベルセットに関して、レベルセットオブジェクトとカットメッシュ、積分法、有限要素法を定義します。

ファイル¶

ファイル

説明

getfem_level_set.h と getfem_level_set.cc

オプションの二次レベル設定関数を使用して、レベルセット関数 (Lagrange fem で定義されたスカラーフィールド) を定義します。

getfem_mesh_level_set.h と getfem_mesh_level_set.cc

1つまたは複数のレベルセットに対してメッシュをカットします。

getfem_fem_level_set.h と getfem_fem_level_set.cc

要素に依存し、1つまたは複数のレベルセットによってカットされる特殊な有限要素メソッドを定義します。

getfem_mesh_fem_level_set.h と getfem_mesh_fem_level_set.cc

1つまたは複数のレベルセットによってカットされた形状関数を持つ mesh_fem オブジェクトを生成します。

getfem_mesh_im_level_set.h と getfem_mesh_im_level_set.cc

レベルセットによってカットされた積分法を表す mesh_im を生成し、レベルセット、他方、レベルセット自体の両側または片側にあります。

getfem_level_set_contact.h と getfem_level_set_contact.cc

インプラントの位置決めを簡単に解析するための、レベルセットベースの大型スライディングコンタクトアルゴリズム。
getfem_convect.h

ベクトルフィールドに対する量の対流を計算します。たとえば、レベルセット関数の進化を計算するために使用されます。Galerkin特性法

状態¶

安定。

視点¶

異なるゾーンを計算するアルゴリズムを明確にします。
