Fem モジュール¶

説明¶

Fem モジュールは GetFEM++ これは、要素レベルと自由度の有限要素を記述します。有限要素法は異なったタイプである場合もあります。それらは幾何学的変形によって同等であるスカラーかベクトル、多項式、区分多項式または非多項式となります。また、自由度の記述は、一般的な方法で2つの隣接要素間の互換性のある自由度を収集することができるようにする必要があります (例えばLagrange 2D 要素が別のLagrange 1D 要素に接続する場合)。

ファイル¶

ファイル

説明

bgeot_poly.h と bgeot_poly_composite.h と bgeot_poly.cc と bgeot_poly_composite.cc

いくつかのクラスは、参照要素の有限要素法の形状関数を記述するために、多項式と区分的な多項式を表現します。

getfem_fem.h と getfem_fem.cc と getfem_fem_composite.cc

有限要素と自由度の記述子。 getfem_fem.cc で定義される多項式有限要素と getfem_fem_composite.cc の区分多項式有限要素

getfem_fem_global_function.h と getfem_fem_global_function.cc

ユーザーが指定したグローバル関数として定義された基本関数を持つ有限要素法を定義します。特異関数による凝縮やメッシュレス法の実装に有用です。

getfem_projected_fem.h と getfem_projected_fem.cc

(mesh_fem で表される) 別のメッシュ上の有限要素空間の射影である有限要素法を定義します。高汎用アセンブリ言語では、補間変換によってこの機能も提供されていることに注意してください。

getfem_interpolated_fem.h と getfem_interpolated_fem.cc

別のメッシュ上の有限要素空間 (mesh_fem で表される) の補間である有限要素法を定義します。高ジェネリックアセンブリ言語では、補間変換によってこの機能も提供されていることに注意してください。

状態¶

2つのファイル getfem_fem.cc と getfem_fem_composite.cc は主に基本的な要素のすべての有限要素の説明が含まれています。定義された有限要素の完全なリストは 付録A.有限要素法リスト で与えられます。

いくつかの他のファイルのようないくつかの特定の有限要素を定義する getfem_fem_level_set.h は1 つまたは複数のレベルセットで既存の要素を “切断” することができる複雑な構造です。

自由度を記述する方法は、グローバルに必要事項を満たしますが (一般的な方法で別の要素から接続された自由度) が、少しあいまいで、あまりにも複雑です。

逆に、補助行列 M で非等価要素を表現する方法は、いくつかの要素 (Hermites 要素、Argyrisなど) でその効率性が証明されています。

展望¶

このモジュールの主な不満は、自由度の説明について完全に満足していないということです。自由度の記述が一時的であるために考案されたので、A から Z までの要素を構築する方法についての1つのドキュメンテーションが今のところ作られなかった主な理由です。設計の努力はこうして完全に自由度の記述に関してこのモジュールを主として安定させるために提供されるべきであるまた多分立体求積法に同様の方法で部分的に外部化であることができる有限要素の記述メソッドは、少なくとも最も単純な有限要素(等価および多項式有限要素)です。
