MeshFem モジュール¶

説明¶

MeshFem モジュールは、特定のメッシュに対する有限要素法 (空間) を表すことを目的としています。mesh_fem オブジェクトは、指定されたメッシュに永続的にリンクされ、メッシュの変化 (特に要素の追加または削除) に反応することができます。mesh_fem オブジェクトは、もちろん、すべての要素が同じ有限要素メソッドを共有していることが最も一般的なケースであっても、メッシュの各要素に異なる有限要素メソッドを関連付けることができます。

ファイル¶

ファイル

説明

getfem_mesh_fem.h と getfem_mesh_fem.cc

メッシュ全体の有限要素を表す構造体を定義します。メッシュの各要素は、有限要素法に関連付けられています。これは、実行する非常に複雑な構造である dof 識別とナンバリングは、グローバルなリニアリダクションを可能にします。

getfem_mesh_fem_global_function.h と getfem_mesh_fem_global_function.cc

fem_global_function として定義された fem で mesh_fem を定義します。リンクされた fem_global_function の基本関数のリストを更新するための便利なメソッドを提供します。

getfem_mesh_fem_product.h と getfem_mesh_fem_product.cc

2つの有限要素法の直積の一種である mesh_fem オブジェクトを生成します。Xfem 充実に役立ちます。

getfem_mesh_fem_sum.h と getfem_mesh_fem_sum.cc

2つの有限要素法の直和の一種である mesh_fem オブジェクトを生成します。Xfem の充実に役立ちます。

getfem_partial_mesh_fem.h と getfem_partial_mesh_fem.cc

dofの数が減少した mesh_fem を生成します。

getfem_interpolation.h と getfem_interpolation.cc

異なるメッシュ間の2つの有限要素メソッド間の補間。代わりに、高レベルのジェネリックアセンブリの補間機能を使用できます。
getfem_derivatives.h

(不連続) Lagrange有限要素における有限要素場の微分の補間代わりに、高レベルのジェネリックアセンブリの補間機能を使用できます。

getfem_inter_element.h と getfem_inter_element.cc

要素間の計算のためのフレームワークを作る試み (例えば通常の微分でジャンプ)。継続中で、おそらくジェネリックアセンブリ言語に統合されています。

getfem_error_estimate.h と getfem_error_estimate.cc

誤差推定の計算のための枠組みを作る試み。継続中で、おそらくジェネリックアセンブリ言語に統合されています。
getfem_crack_sif.h

SIF (応力拡大係数) の計算のためのクラックサポート機能。

getfem_torus.h と getfem_torus.cc

2次元構造を半径寸法で拡張する mesh_fem オブジェクトを適応します。

状態¶

安定しています。あまり進化していません。

展望¶

並列化の自由度のナンバリングが行われます。最適 (単純な) アルゴリズムが存在します。
